Tentang Aplikasi

Aplikasi Quiz Berbasis Wesite ini dibuat untuk memenuhi tugas kuliah project akhir dari mata kuliah Rekayasa Perangkat Lunak. Adapun perencanaan dalam proses pembuatan Aplikasi Quiz Berbasis Website, seperti Menentukan tempat studi kasus, Mengumpulkan data, Melakukan survei, Analisa, Desain, Implementasi dan Pengujian
